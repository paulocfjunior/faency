## Examples

### Avatar using non-existing image with fallback to initials

```js live
<Avatar src="https://avatars0.githubusercontent.com/u/invalid-user">KS</Avatar>
```

### Avatar using non-existing image with fallback to custom initials

```js live
<Avatar src="https://avatars0.githubusercontent.com/u/invalid-user">
  <Text textColor="white" marginBottom={1}>
    K
  </Text>
  <Text textColor="white" marginTop={1}>
    S
  </Text>
</Avatar>
```

### Avatar using existing image on a dark variant

```js live
<Box backgroundColor="dark" padding={2}>
  <Avatar
    src="https://images.unsplash.com/photo-1579380287268-aa88d096651c?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=400&q=80"
    backgroundColor="purple"
    variant="dark"
  >
    KS
  </Avatar>
</Box>
```

### Avatar using non-existing image with fallback to initials on a dark variant

```js live
<Box backgroundColor="dark" padding={2}>
  <Avatar src="https://avatars0.githubusercontent.com/u/invalid-user" variant="dark">
    KS
  </Avatar>
</Box>
```
